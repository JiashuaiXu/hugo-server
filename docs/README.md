# Hugo 博客功能配置文档

本目录包含 Hugo 博客所有自定义功能的详细配置文档。

## 📚 文档索引

### [01-search-feature.md](./01-search-feature.md)
**站内搜索功能**
- 添加全站搜索功能
- 支持文章标题、内容、标签搜索
- 使用 PaperMod 主题内置的 Fuse.js 实现

### [02-reading-progress-bar.md](./02-reading-progress-bar.md)
**顶部阅读进度条**
- 页面顶部显示阅读进度
- 实时响应滚动位置
- 紫色渐变样式

### [03-sidebar-toc.md](./03-sidebar-toc.md)
**目录侧边栏**
- 桌面端固定在右侧
- 移动端自适应布局
- 支持点击跳转

### [04-visitor-statistics.md](./04-visitor-statistics.md)
**访问统计功能**
- 博客总访问量和访客数
- 单篇文章阅读数
- 使用不蒜子统计服务

## 📋 每个文档包含

- ✅ **功能说明**: 功能的作用和效果
- 🔧 **配置步骤**: 详细的实现步骤
- 🎨 **自定义配置**: 如何调整样式和设置
- 🗑️ **移除步骤**: 完整的功能移除方法
- ⚠️ **注意事项**: 使用中的常见问题
- 🔍 **故障排除**: 问题诊断和解决方法

## 🎯 使用方式

1. **添加功能**: 按照对应文档的"配置步骤"操作
2. **自定义**: 参考"自定义配置"章节调整
3. **移除功能**: 按照"移除步骤"完整移除
4. **遇到问题**: 查看"故障排除"章节

## 🔄 保持 jesse-blog 目录整洁

所有功能文档都存放在本目录(`docs/`)下,而不是 `jesse-blog/` 目录中,这样可以:

- ✅ 保持博客内容目录整洁
- ✅ 避免文档被构建到网站中
- ✅ 方便版本控制和管理
- ✅ 每个功能独立,便于维护

## 🚀 快速测试

配置完成后,运行以下命令测试:

```bash
cd jesse-blog
hugo --gc --minify    # 构建
hugo server -D        # 本地预览
```

## 💡 提示

- 每次修改配置后建议重新构建以查看效果
- 访问统计功能在本地开发时可能不显示准确数据
- 建议先在测试分支上实验,确认无误后再合并
